通过Hexo和github_Page来搭建一个免费的个人博客

发表于 2021-09-19

首先准备

  • Node_JS

  • Git

  • 安装Hexo

  • 安装npm和cnpm

检查是否装好

  • git ：git –version
  • node_js : node -v
  • 其他 ：名字 -v 如：hexo -v

注意事项

  • git的SSH-key问题（非常重要，否则上传不了，第一次使用要配置）

  • 目录问题，如是在blog文件下进行操作还是在themes文件下操作

  • 在预览后记得CTRL + C退出，否则会出错

  • Node_JS 的环境变量配置

  • Hexo的版本注意（不同的版本的设置略微不同，如要求ssh还是http）

  • 同上，我的Hexo是4.3.0 在最后的设置就要是 git@github.com……..而不是https://github.com……..

一些需要记得的常用Hexo命令

  • hexo d : 上传博客（deploy）
  • hexo s : 预览（记得CTRL + C）
  • hexo clean : 清除缓存
  • hexo new “….” : 新建一个博文

自己遇到的一些”坑”

  • 换了个next theme后，执行任何hexo命令后都会得到一个Error提示：

    Cannot find module ‘hexo-util’ ： 运行一个命令安装下就可以：npm install hexo-util –save
